# 外贸文档与模板

本仓库包含外贸必备知识文档与可复用模板（中英双语）。主要文件：

- docs/外贸必备知识_中英.md — 外贸必备知识（中英双语指南）
- docs/templates/Contract_Template_CN_EN.md — 合同/销售合同模板（中英）
- docs/templates/Invoice_Template_CN_EN.md — 商业发票模板（中英）
- docs/templates/Quotation_Template_CN_EN.md — 报价单模板（中英）

使用建议：
- 在开始与客户沟通前，先确认模板中的占位符（如公司名、币种、港口、日期等）已被替换为实际值。
- 根据公司政策调整付款条款、检验标准与争议解决条款。

如需我把模板拆成单独的中/英文文件、增加更多模板（如装箱单、原产地证格式、信用证样例）或将 README 翻译成英文，请告诉我。
